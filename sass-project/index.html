Install Sass

Step 1—

To begin using Sass with using the terminal, type cmd-spacebar and type terminal. Once in the terminal we need to do 1 of 2 things. If this is the first time using Sass on this machine, we need to install Sass. Otherwise, we need to update it. First timers, please type sudo gem install sass into the command line. This will go out and find the gem sass and install it on your computer. It will ask you for your password, this is just your computers login password.

For those of us who have already installed Sass and need to get going on a project, update Sass by typing sudo gem update sass.

To check the current version of Sass or to simply make sure that it's installed, type sass -v

Set up a Project

Step 2—

To get the Sass compiler up and running, watching our project, we will need to use the --watch command, so that it can begin looking for changes in our project.

First things first! We need to make sure that we're in the correct directory of our project. Here are a couple helpful commands in order to get us to that point;

  • Change directory to desktop > project-name cd desktop/project-name
  • If you're already on the desktop its even more simple, just cd project-name
  • Other helpful command line shortcuts are; pwd, which shows you where you are at...
  • ...and ls will show me what files are in the current directory im in.
  • To stop watching type Ctrl-C

Once you know you're in the correct file of that project that you're going to be working in, it's probably best to keep Sass && CSS in separate folders and/or directories. Therefore, we'll want to use the command sass --watch sass:css that will say "hey Sass, watch all the files in the 'sass' folder for changes and update the stylesheet in the css directory." A few other cool tips to note are;

  • You can start watching a file before a file has been created or after a file has been created.
  • Also, a folder for css doesn't need to be created intially. If you didn't want to create the folder before hand, once you start watching a file and typing the sass --watch sass:css command, it will auto generate a css directory w/ the css file once a change has been made.
  • Enhance and compress your compiled code easily by using the command sass --watch --style compressed sass:css
  • Likewise, you can watch just a single file by typing sass --watch --style compressed style.sass:style.css
  • Important: Dw will ask you anytime the Sass file has been changed that the css file has been modified outside of Dw and if you would like to update/save those changes. You want to make sure that you click Yes and check for it to Stop asking you.
  • Another useful tip is that the command line sometimes just kinda goes invisible. I'm not really sure what that is, but Sass is still compiling and working. If you type shift-command-period you can check where you're at pwd and start typing commands again =)
